This program was written to safely update the firmware of a T+A DAC8DSD.
It was not intended to serve as an example of good programming practice.

Who so ever wants to improve this program or increase its range of use is 
welcome to do so....

The program can be compiled with Lazarus IDE v 1.6
It uses the serial communication unit synaser which can be downloaded from 
this site: http://synapse.ararat.cz/doku.php/download
